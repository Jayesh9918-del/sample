# **App Name**: Architechs.io

## Core Features:

- Showcase Portfolio: Display featured architectural projects with high-quality images and detailed descriptions.
- Architect Directory: Provide a searchable directory of architects, with profiles, contact information, and project history.
- Blog/News Section: Publish articles, news, and insights related to architecture, design, and urban planning.
- AI-Powered Design Ideas: Use generative AI to create an initial range of relevant design ideas based on user inputs (location, materials, style). The user refines the generated ideas through a tool to filter results and add extra specifications.

## Style Guidelines:

- Primary color: Dark slate gray (#334756) for a professional and sophisticated look.
- Background color: Light gray (#F0F4F8) for a clean, modern backdrop.
- Accent color: Muted teal (#82A3A1) for subtle highlights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, offering a clean, modern and objective appearance suitable for both headlines and body text.
- Use minimalist line icons to represent different architectural styles, services, and features.
- Employ a grid-based layout with ample whitespace for a clean, organized, and visually appealing presentation.
- Incorporate subtle animations and transitions to enhance user engagement and provide a smooth browsing experience.
